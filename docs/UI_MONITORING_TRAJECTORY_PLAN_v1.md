# UI: мониторинг траектории (план v1)

Дата: 2026-03-06

## Что уже сделано

- Настройки `PlutoPlayer` перенесены на вкладку `Мониторинг` в layout `l_gpsSDRSim_options_m`.
- На вкладке `Сценарий` в `l_SDR_options` оставлен блок `GPS SDR Sim` (эфемериды, статика, старт/финиш).
- В `vl_trajectory_visualization_m` добавлен отдельный 3D-виджет мониторинга.
- После успешной подготовки сценария запускается анимация движения маркера по `trajectory.csv` по локальному времени (циклически).
- Если массив точек в памяти недоступен, траектория подгружается асинхронно из `last_run_dir/trajectory.csv`.

## Что планируется сделать далее

### 1) 3D-анимация в `vl_trajectory_visualization_m`

- Статус: `DONE (v1 базовая реализация)`.
- Источник данных: `trajectory.csv` из подготовленного сценария.
- Режим анимации: по локальному времени, без обратной связи от SDR.
- Управление (план следующего шага):
  - `Старт/Пауза`
  - `Сброс на t=0`
  - Ползунок позиции по времени
  - Множитель скорости воспроизведения (`x0.5`, `x1`, `x2`)
- При отсутствии траектории: показывать понятный placeholder и не запускать таймер анимации.

### 2) телеметрия в `gl_trajectory_params_m`

- Выводить в реальном времени:
  - текущую скорость
  - текущие координаты (`X, Y, Z`)
  - текущую высоту
  - пройденное расстояние
- Расчет выполнять по траектории (интерполяция по времени анимации), без внешних датчиков.
- Обновление синхронизировать с тиком анимации.

## Ограничения v1

- Нет верификации «фактической позиции» от SDR/маяка.
- Нет коррекции тайминга по внешнему источнику; только локальный UI-clock.
