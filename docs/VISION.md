# VISION

## Goal
Создать расширяемую операторскую оболочку для управления:
- внешними exe программами
- подключенными устройствами
- сценариями работы стенда

Система должна легко расширяться добавлением новых сервисов
без изменения существующих.

---

## Non-Goals

- Не real-time ОС
- Не микросервисная облачная система
- Не попытка заменить драйверы устройств

---

## Typical Use Case

Оператор:
1) выбирает профиль
2) нажимает Start
3) система выполняет сценарий
4) получает телеметрию
5) нажимает Stop

---

## Success Criteria

- Новое устройство подключается добавлением сервиса
- UI не меняется при добавлении сервиса
- Система не блокируется при зависшей exe
