# Численная модель пространственного движения артиллерийского снаряда

## Назначение проекта

Данный проект реализует численное моделирование пространственного движения артиллерийского снаряда как твёрдого тела с учётом:

* поступательного движения с аэродинамическим сопротивлением;
* вращательного движения и стабилизирующего аэродинамического момента;
* кинематики ориентации (уравнения Пуассона);
* стандартной атмосферы ISA (0–32 км);
* численного интегрирования методом Эйлера первого порядка;
* автоматической верификации корректности модели.

## Конфигурация расчёта

Все параметры эксперимента задаются в файле `vkr_config.json`, включая:

* параметры снаряда (масса, площадь, коэффициенты);
* параметры вращения и стабилизации;
* начальные условия (скорость, углы, высота);
* параметры численного интегрирования.

Файл является **каноническим источником параметров** и используется по умолчанию при запуске.

---

## Запуск моделирования

### 1. Численный расчёт (без построения графиков)

```bash
python run_vkr.py --config vkr_config.json --out outputs
```

В результате формируются файлы:
- `outputs/trajectory.csv` — траектория и состояние снаряда;
- `outputs/diagnostics.csv` — диагностические величины;
- `outputs/run.json` — сводная информация о запуске и точке удара.

### 2. Построение графиков

```bash
python visualization.py --out outputs
```

Графики сохраняются в каталог `outputs/plots/`.

## Архитектура вычислений

Проект разделён на два логически независимых этапа:

1. Численный расчёт (`run_vkr.py`) — интегрирование уравнений движения.
2. Визуализация (`visualization.py`) — постобработка и построение графиков.

Такое разделение обеспечивает воспроизводимость расчётов и возможность
использования вычислительного ядра как самостоятельного сервиса.

## Верификация модели

Для проверки корректности численной реализации используется автоматический чек-лист:

```bash
python selfcheck.py
```

В ходе проверки контролируются:

* корректное завершение интегрирования;
* отсутствие NaN / Inf;
* сохранение ортонормальности матрицы ориентации;
* корректность геометрии подъёмной силы;
* построение всех требуемых графиков.

Результаты проверки сохраняются в `selfcheck/selfcheck_report.json`.

---

## Используемые допущения

* Атмосфера соответствует стандартной модели ISA (0–32 км);
* Численный метод — явный метод Эйлера первого порядка;
* Физическая модель не упрощается по сравнению с математической постановкой;
* Численная стабилизация допускается только для матрицы ориентации.
